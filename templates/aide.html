1. Introduction à Hilo

Hilo est une application personnelle de suivi de la tension artérielle. Elle vous permet d'importer vos relevés depuis des fichiers PDF, de les analyser sous forme de graphiques, de suivre vos traitements et de générer des rapports pour vos médecins.

Ce que Hilo vous permet de faire

  • Importer automatiquement vos mesures depuis des PDFs de tensiomètre

  • Visualiser vos tendances systolique, diastolique et fréquence cardiaque

  • Suivre vos objectifs tensionnels fixés avec votre médecin

  • Gérer vos traitements médicamenteux

  • Réaliser des protocoles d'automesures (règle des 3 × 3)

  • Exporter et imprimer des rapports détaillés

  • Suivre l'évolution de votre poids

💡 Hilo fonctionne entièrement en local sur votre ordinateur. Vos données ne sont jamais envoyées sur Internet. (sauf transfert FTP)

Étape 2 — Cibles tensionnelles

Saisissez ici les objectifs tensionnels recommandés par votre médecin. Ces valeurs servent de référence dans tous les graphiques et rapports.

Cible Valeur par défaut Description
Systolique (mmHg) 135 Pression maximale lors de la contraction du cœur
Diastolique (mmHg) 85 Pression minimale lors du relâchement du cœur
⚕️ Ces valeurs sont indicatives. Demandez toujours conseil à votre médecin pour définir vos cibles personnelles.

Étape 3 — Dossier de données

Hilo stocke toutes vos données dans un fichier local (hilo.db). Choisissez un dossier sur votre ordinateur où ce fichier sera créé. Des suggestions de dossiers sont proposées automatiquement.

  • Bouton Parcourir : ouvre le sélecteur de dossier de votre système

  • Suggestions : cliquez sur l'un des chemins proposés pour le sélectionner directement

Étape 4 — Import initial (optionnel)

Si vous possédez déjà un fichier hilo.csv exporté depuis une version précédente, vous pouvez l'importer ici pour récupérer vos données existantes.

  • Bouton Choisir le fichier : sélectionnez votre hilo.csv

  • Bouton Passer : ignorez cette étape si vous n'avez pas de fichier à importer

Modal de saisie d'une séance

En cliquant sur une cellule, un formulaire s'ouvre avec deux colonnes :

Colonne Description
Bras prioritaire (bleu, bordure pleine) Sys / Dia / FC obligatoires

Rapport de protocole

Après clôture (ou consultation de l'historique), le rapport présente :

  • Tableau détaillé par séance avec moyennes par moment (Matin/Soir) et globales

  • Valeurs du bras secondaire affichées en italique entre parenthèses : ex. 135 (132)

  • Classification ESH des moyennes

  • Écart par rapport à vos cibles

  • Synthèse globale avec comparaison bras principal / bras secondaire

9. Questions fréquentes

Mon PDF n'est pas reconnu

Hilo supporte les PDFs générés par les tensiomètres connectés compatibles. Vérifiez que le fichier n'est pas protégé par un mot de passe et qu'il contient bien des données tabulaires de tension.

Comment mettre à jour mon poids ?

Allez dans Paramètres → Profil & Cibles, modifiez le champ Poids, puis cliquez sur Enregistrer. Le nouveau poids sera automatiquement ajouté à l'historique et la courbe de l'Accueil sera mise à jour.

Puis-je avoir des mesures des deux bras ?

Oui, dans le protocole d'automesures. Le modal de saisie propose deux colonnes : bras principal (obligatoire) et bras secondaire (optionnel). Les valeurs du bras secondaire apparaissent en italique entre parenthèses dans le rapport.

Comment sauvegarder mes données ?

Dans Paramètres → Base de données, cliquez sur Sauvegarder la base. Une copie horodatée est créée dans le même dossier que votre base hilo.db. Vous pouvez également exporter toutes vos mesures en CSV.

Mes données sont-elles sécurisées ?

Hilo fonctionne entièrement en local. Aucune donnée n'est envoyée sur Internet, sauf si vous activez volontairement l'export FTP vers votre propre serveur.

Comment partager le Dashboard avec mon médecin ?

Utilisez l'export local (Paramètres → Export) pour générer un fichier HTML autonome. Ce fichier peut être envoyé par email ou partagé sur une clé USB — il s'ouvre dans n'importe quel navigateur sans installation.
